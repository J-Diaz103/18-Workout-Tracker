# 18-Workout-Tracker

## Description

This application will be able to keep track of your workouts for a week.
You can keep track of the type of workout and the Reps, weight, duration and distance.

## Languages

- HTML
- CSS
- JavaScript
  - Libaries and Dependencies
  - Express
  - MongoDB
  - Mongoose
  - Heroku
  - MongoDB Atlas
